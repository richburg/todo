
# todo

Manage your tasks through the CLI.


## Installation

See https://pipx.pypa.io/latest/installation/ and follow the guide below:

```bash
git clone https://github.com/richburg/todo
cd todo/
pipx install .
```

## Documentation

`todo --help` or `todo [specific command] --help`.
